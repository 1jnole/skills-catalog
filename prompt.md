Objetivo: comprobar que OpenSpec funciona en este repo sin crear cambios nuevos.

Invoca:
- $spec-spec-lint
- $spec-tasks-lint

Pasos:
1) Lista slugs existentes en openspec/changes/ y elige uno representativo.
2) Ejecuta $spec-spec-lint contra openspec/changes/<slug>/specs/spec.md.
3) Ejecuta $spec-tasks-lint contra openspec/changes/<slug>/tasks.md.
4) Ejecuta npm run verify y confirma PASS.

Entrega:
- El slug usado
- Resultado de ambos lints
- Evidencia del comando npm run verify (salida recortada)
