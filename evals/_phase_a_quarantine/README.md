# Phase A Quarantine

This folder isolates historical or inherited eval artifacts that should remain recoverable but must not be mistaken for the active Promptfoo runtime surface.

Rules:
- files moved here are historical context, migration residue, or pending review artifacts
- files here are not part of the active runtime boundary
- active Promptfoo configs and suites stay under `evals/engines/promptfoo/`
- active local authoring artifacts stay under `evals/cases/`
