# Example Project Brief

Build a small web app with:
- A list view of `items` with sort by `name` and `createdAt`
- A detail view for a selected item
- Deterministic "item of the day" based on current date (no randomness)

Constraints:
- Provide a single verification command: `npm run verify`
- Do not invent external APIs; use a local JSON file if needed
- Prefer small, reviewable changes
