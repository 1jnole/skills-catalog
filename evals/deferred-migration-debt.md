# Deferred Migration Debt

These items are intentionally outside the completed migration closeout:

- `previous-skill` baseline support
- a second engine or multi-engine orchestration
- extra observability or reporting beyond the current local artifacts
- broader CI automation for the new flow
- final deletion of historical Laminar source code once no compatibility value remains

They are backlog items after the migration, not unfinished acceptance criteria for the closed harness move.
